# Weather App (In Class)

A simple, real-time weather application built with HTML, CSS, and JavaScript.

## Features

- **Real-time Weather Data**: Fetches current weather information using the WeatherAPI.
- **Location Search**: Search for weather conditions in any city.
- **Dynamic UI**: Updates temperature, city name, date, time, and weather condition icon dynamically.
- **Responsive Design**: Clean and simple user interface.

## Technologies Used

- **HTML5**: For structure.
- **CSS3**: For styling (Flexbox, custom fonts).
- **JavaScript (ES6+)**: For fetching API data and DOM manipulation.
- **WeatherAPI**: External API for weather data.

## How to Use

1.  Clone the repository.
2.  Open `index.html` in your browser.
3.  Enter a city name in the search bar and click "Search".

## Setup

No build steps required! Just open `index.html` to run the app.
